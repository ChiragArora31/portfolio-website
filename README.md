# Portfolio Website

A clean, minimalistic, and modern portfolio website showcasing my work as a software engineer. Built with vanilla HTML, CSS, and JavaScript.

## Features

- 🎨 **Glassmorphism Design** - Modern frosted glass effects with backdrop filters
- 🌓 **Dark/Light Theme** - Seamless theme switching with localStorage persistence
- 📊 **Scroll Progress Indicator** - Visual feedback of reading progress
- ♿ **Accessible** - WCAG AA compliant with skip links and ARIA labels
- 📱 **Fully Responsive** - Works flawlessly on all devices and screen sizes
- ⚡ **Optimized Performance** - Lightweight codebase (~65KB total)
- 🔍 **SEO Ready** - Complete Open Graph and Twitter Card meta tags

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, backdrop-filter, animations
- **Vanilla JavaScript** - No frameworks, just clean ES6+
- **Google Fonts** - Inter typeface

## Sections

- **Hero** - Introduction and key details
- **Blog** - Featured blog posts
- **Projects** - Showcase of key projects with GitHub links
- **Footer** - Social media and contact links

## Installation

1. Clone the repository:
```bash
git clone https://github.com/ChiragArora31/portfolio-website.git
cd portfolio-website
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

3. Visit `http://localhost:8000` in your browser

## Deployment

This website is static and can be deployed to:
- **Netlify** - Drag and drop the folder
- **Vercel** - Import from GitHub
- **GitHub Pages** - Enable in repository settings
- **Any static hosting service**

## Customization

1. **Update personal info** in `index.html`
2. **Modify colors** in `styles.css` CSS variables
3. **Add/remove projects** in the Projects section
4. **Update meta tags** with your actual domain (line 14 in index.html)
